# 函数式编程 (Functional Programming)

函数式编程是一种编程范式，它将计算视为数学函数的求值。

## 核心特性

1. **纯函数** - 相同输入永远返回相同输出，无副作用
2. **不可变性** - 数据一旦创建就不能修改
3. **高阶函数** - 函数可以作为参数和返回值
4. **声明式** - 描述"做什么"而非"怎么做"

## 优势

- 更容易测试
- 更容易推理
- 更容易并发
- 更少的 bug
