# Qubit

这是一个现代化的个人知识分享平台，整合了多个模块：

## 核心功能

- **博客系统** - 文章发布、分类标签、评论系统
- **数字花园** - 双向链接、知识图谱可视化
- **项目展厅** - 作品展示、技术栈标注
- **书签收藏** - 外部链接管理
- **书评影评** - 评分和评价记录
- **时间线** - 个人里程碑记录

## 技术亮点

- Next.js 16 App Router
- TypeScript 类型安全
- Tailwind CSS 4
- MDX 内容支持
- Cloudflare Pages 部署
