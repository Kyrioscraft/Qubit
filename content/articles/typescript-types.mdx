# TypeScript 类型系统深入理解

TypeScript 的类型系统是其最强大的特性之一。本文将深入探讨其核心概念。

## 泛型基础

泛型允许我们创建可重用的组件，这些组件可以与多种类型一起工作：

```typescript
function identity<T>(arg: T): T {
  return arg;
}

// 使用
const result = identity<string>("hello");
```

## 类型推断

TypeScript 的类型推断能力非常强大：

```typescript
let x = 3; // 推断为 number
let y = [1, 2, 3]; // 推断为 number[]
```

## 高级类型模式

### 条件类型

```typescript
type IsString<T> = T extends string ? true : false;

type A = IsString<string>; // true
type B = IsString<number>; // false
```

### 映射类型

```typescript
type Readonly<T> = {
  readonly [P in keyof T]: T[P];
};
```

## 实践建议

1. 优先使用 `interface` 定义对象形状
2. 使用 `type` 定义联合类型和复杂类型
3. 避免使用 `any`，优先使用 `unknown`
