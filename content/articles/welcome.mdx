# 欢迎使用 Qubit

Qubit 是一个现代化的个人知识分享平台。它超越了传统博客的概念，整合了多个模块，打造一个完整的知识生态系统。

## 核心功能

### 博客系统

- Markdown/MDX 支持
- 分类与标签系统
- 文章系列
- 阅读时间估算
- 自动生成目录

### 数字花园

- 知识节点互联（双向链接）
- 可视化知识图谱
- 内容成熟度标记

### 其他模块

- 项目展厅
- 书签收藏
- 书评影评
- 时间线

## 设计理念

Qubit 的设计遵循以下原则：

1. **知识中枢** - 系统化整理和分享个人知识
2. **作品展厅** - 展示项目和成果
3. **思想花园** - 数字花园式的互联笔记
4. **社交节点** - 与读者建立真实连接

## 技术栈

- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4
- MDX (Content Collections)
